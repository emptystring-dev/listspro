# Lists Pro

Public downloads and update artifacts for Lists Pro, a self-hosted multi-column X list
dashboard.

## Download

[Download Lists Pro for Apple Silicon macOS](https://github.com/emptystring-dev/listspro/releases/latest/download/stable-macos-arm64-ListsPro.dmg)

The first release is still being prepared. The download link becomes active when the
private source repository publishes `v0.1.0`.

## Install

1. Open the downloaded DMG.
2. Drag Lists Pro to Applications.
3. Open Lists Pro and follow the setup instructions.

Releases are Developer ID signed, notarized by Apple, and updated automatically. This
repository contains public binaries and updater metadata only; source code is maintained
separately.
